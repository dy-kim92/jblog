메인페이지 
	- 로그인, 비로그인 시 로고 밑 메뉴 변경
	- 로그인 시 : 로그아웃 / 내블로그
	- 비로그인 시 : 로그인 / 회원가입

로그인 페이지
	- ID 혹은 PASSWORD 틀릴 경우 알럿 출력 후 리다이렉트

회원가입 페이지
	- 이름 3자 이상
	- 비밀번호 4자 이상
	- 중복체크 해야 가입 됨
	- 약관동의 해야 가입 됨
	
블로그 메인
	- HEADER 메뉴
		- 본인의 블로그일 시 내 블로그 관리 노출
		- 로그인 / 비로그인에 따라 로그인 / 로그아웃 으로 버튼 노출
		- 블로그명은 내블로그관리 -> 블로그 제목에 설정한 블로그 제목으로 노출됨
		
	- FOOTER
		- 블로그명은 내블로그관리 -> 블로그 제목에 설정한 블로그 제목으로 노출
		- 카피라이트 표시
		
	- 카테고리 메뉴 & 로고이미지
		- 내 블로그 관리 -> 로고이미지 업로드한 파일로 노출됨 (카테고리 가로크기를 넘어가지 않음)
		- 해당 블로그에 등록된 카테고리 목록 노출
	
	- 메인 포스트
		- 가장 최근에 게시된 포스트 노출
		- 전체 포스트가 없을 경우 미노출
		
	- 메인 코멘트
		- 가장 최근에 게시된 포스트가 있을 경우 노출
		- 코멘트 리스트는 없을 경우 미노출하고 입력란만 노출됨
		- 코멘트의 경우 로그인한 authUser의 코멘트에만 삭제버튼 노출됨
		
	- 메인 리스트
		- 카테고리 구분 없이 전체 포스트 리스트가 노출됨
	
	- 카테고리 선택 시 
		- 해당 카테고리 최신 게시물이 메인포스트로 노출
		- 해당 카테고리 포스트 리스트 노출
		- 코멘트 리스트는 없을 경우 미노출하고 입력란만 노출됨
		
	- 포스트 선택 시 
		- 해당 포스트 내용 노출
		- 코멘트 리스트는 없을 경우 미노출하고 입력란만 노출됨
		- 포스트 리스트는 해당 포스트의 카테고리 안의 포스트목록만 노출됨
		
내 블로그 관리
	- 홈 : 블로그 홈으로 이동
	
	- 기본설정
		- 블로그 제목의 기본값은 현재 블로그의 제목으로 입력되어 노출됨
		- 노출된 로고이미지는 업로드한 이미지가 아닌 현제 등록된 이미지
		- 업로드할 로고 미선택 시 현제 이미지 그대로 유지
	
	- 카테고리
		- 카테고리 리스트에 최신 카테고리가 상단에 위치하게 노출됨
		- DB등록된 카테고리 번호, 이름, 등록된 포스트 수, 설명, 삭제버튼이 노출됨
		- 삭제 기능은 카테고리 내에 포스트가 작성되어 있을 경우 알럿메세지 노출
		- 카테고리명, 설명 미입력 시 알럿 노출 후 포커스 처리
		- 등록 성공 시 성공알럿 노출 후 페이지 리다이렉트
		
	- 글쓰기
		- 제목, 내용 미입력 시 알럿 노출 후 포커스 처리
		- 등록 성공 시 성공알럿 노출 후 페이지 리다이렉트
		- 제목 우측 Dropdown List 에 등록된 카테고리 목록 선택하여
		  선택된 카테고리에 포스트 등록
